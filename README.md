# uTools Mirrors 插件

一个支持常见开发环境代理一键搜索与粘贴的 uTools 插件。

## 功能
- 支持 npm、go、pip、yum、flutter 等常见代理的镜像源一键复制
- 支持镜像源搜索、快速定位
- 支持还原官方源
- 适配 uTools 全局搜索

## 技术栈
- React
- TypeScript
- TailwindCSS
- shadcn/ui

## 使用方法
1. 在 uTools 插件市场安装本插件
2. 输入相关关键词（如"npm 代理"）即可快速搜索和复制命令

## 开发
```bash
pnpm install
pnpm run dev
```
